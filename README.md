# NM3216 Project 2 : Falling of the Shield Hero!

A Top-Down Dungeon Crawler where you play as Delta, a Shield-User to fight monsters in real time.

# Controls

WASD to walk.
SPACEBAR to dash.
LEFT_CLICK to shoot.
RIGHT_CLICK to parry.
Keep holding RIGHT_CLICK to block.
